# NAME

GraphQL - Perl implementation of GraphQL

# SYNOPSIS

Perhaps a little code snippet.

    use GraphQL;
    my $foo = GraphQL->new();

# DESCRIPTION

See [GraphQL::Type](https://metacpan.org/pod/GraphQL::Type) for description of how to create GraphQL types.
